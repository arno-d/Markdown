## Variáveis
> As variáveis são espaços de memória que tem como objetivo guardar/armazenar informações.

1. Números:
    - int
    - double
    - float
1. Textos
    - string

1. Booleanos
    - bool
1. Listas
    - Listas
    - List<>
    - int[]

## Condições
> if, switch, ternários

<!-- somente 1 * faz a palavra ficar em Italico
2 * faz a palavra ficar em Negrito 
3 * faz ambos (Italico e Negrito)
1 ` faz a palavra inserida dentro ficar em destaque, com a cor cinza ao redor
3 ``` faz a palavra inserida ficar em destaque, em uma cor mais escura (talvez preto)
o _ faz a mesma função do *, nas mesmas quantidades (ou seja, 1 _ faz o mesmo que 1 *, e assim em diante)-->

Condições é o *método* para **validar** se uma ***informação*** é verdadeira, caso seja _verdadeira_ a __condição__ ___aplicada___ `if` executa somente o código que está dentro dele, com isso temos condições para que o código execute somente o necessário 

```csharp
if (valor1 == valor2)
{
    MessageBox.Show("Os numeros são iguais");
}
```
